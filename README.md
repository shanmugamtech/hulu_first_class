## What you get
- Next.js 14 App Router + TypeScript scaffold
- Tailwind CSS config
- Example server/client components (homepage + dynamic movie detail)
- TMDB fetch helper (server-side)
- AI_Report.md template

## Quick start
1. Copy files into a new folder and open in VS Code.
2. I Created `.env.local` with `TMDB_API_KEY=MY_KEY`.
3. Run `pnpm install` or `npm install`.
4. Run `pnpm dev` or `npm run dev`.

## Deploy
- Push to GitHub then import to Vercel.
- Add `TMDB_API_KEY` in Vercel Environment Variables.
